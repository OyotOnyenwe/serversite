About Onyenwe Server

The server is self-hosted and is directly managed. It is a dedicated machine running Ubuntu Server that is primarily accessed remotely.

Self-hosting allows for us to have direct control over plugins and greatly improves server performance. One important strength of self-hosting is improved update times. For the 1.17 update, Onyenwe Server was up under an hour after the update's release, allowing users to experience the update as soon as it came out right here on the server.

In order to stay up to date with the server, consider joining the Discord, subscribing to the YouTube channel, or following the server's Twitter or Instagram accounts.

Specifications

If you were wondering how the server runs, the following is a list of its specs.

  • AMD Ryzen 5 2600X Processor
  • 32GB of 3200MHz RAM
  • Ubuntu Server 20.04.2 LTS
  • OpenJDK 16.0.1
