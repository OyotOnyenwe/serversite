Server IP: mc.onyenwe.com:7777

To play on Onyenwe Aether, you must install the Aether mod on your client. Download links are below. The downloads on this page are direct links to the mods hosted on their official sites. Just click, and downloads will begin.

Modrinth Installation (Recommended)

For easy installation, use the Modrinth client & the premade Onyenwe modpack. Simply download and set up the Modrinth app, then install Forge, then download the .mrpack file and double click it to have the modpack set up for you within the Modrinth launcher automatically.

Install these three things, in this order.

Manual Installation (optional)

Currently, the required client specifications to join the server are:

  • Forge 47.2.20 for Minecraft 1.20.1
  • Aether Mod 1.1.0 for Forge 1.20.1

To set up your client, run the Forge installer (.jar) file (You must have Java installed). Once downloaded, place the Aether jar file in your %appdata%/.minecraft/mods directory.
