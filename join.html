How to Join

Server IP:

mc.onyenwe.com

Joining Policy

Onyenwe Server welcomes all players who wish to join. It is not invite-only or restricted to certain people.

However, the server is not advertised publicly, which means nearly all players in the community join from an invite from someone they know. Any player is welcome to invite anyone they wish.

This approach ensures the healthy growth of the community and mitigates the possibility of raids or hackers publicly advertised servers have to deal with.

Discord Server

It is recommended that all players on the Minecraft server join the Discord server if they have the means.

The invite link is invite.gg/onyenwe.

Feel free to send the invite to any others who play on the Minecraft server or plan to.

Join Now

Java Edition

Windows, macOS, Linux

  • Version 1.17.1
  • Navigate to multiplayer tab and click "Add Server"
  • Set server name to "Onyenwe Server"
  • Set server address to "mc.onyenwe.com"
  • Set server resource packs to "Prompt" (default)
  • Press Done and start playing!

Bedrock Edition

iOS, Android, Windows 10

  • Version 1.17.x (Latest)
  • Navigate to the Servers tab of the game's menu
  • Scroll down to the "Add Server" button
  • If the button is not there, your device does not support custom servers. See below for a workaround.
  • Set server name to "Onyenwe Server"
  • Set server address to "mc.onyenwe.com"
  • Set port to 19132 (default)
  • Press Save and start playing!

BedrockConnect

Xbox, Nintendo Switch, PlayStation

Though 3rd-party servers are not natively supported on these platforms, a workaround is available for Xbox, Nintendo Switch, or PlayStation users that wish to connect.

Follow the tutorial for your device found on this wiki page.

  • Version 1.17.x (Latest)
  • Select "Connect to a Server"
  • Set server address to "mc.onyenwe.com"
  • Set port to 19132 (default)
  • Enable the "Add to Server List" option
  • Press Save and start playing!

Disclaimer: BedrockConnect is by no means a perfect solution, and it is recommended that you understand what it is and how to disable it if it stops working or interferes with other functionality on your device.
