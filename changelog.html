Changelog

Current Version: 1.1.5

v0.1.4 Alpha 11/17/2020
  • Maximum claim blocks is now 10000 (from 2000)
  • Claim blocks acquired per hour is now 150 (from 100)
  • Players can now use /trapped to escape if they are trapped in a claimed area
  • Enabled a Discord integration with the Minecraft server
  • Earlier fixes
    • fixed the /tpa system
    • Enabled the /msg command
v0.1.3 Alpha 11/15/2020
  • Players on Bedrock are now able to type /geyser offhand to place the item they are currently using into their offhand
  • An issue regarding chest autoclaims has been fixed
  • Chat spam is no longer a bannable offense
v0.1.2 Alpha 11/14/2020
  • Combat logging is now disabled
  • Players can now use a stick to find information about others' claimed blocks
  • Updated permissions
    • /home is now available
    • /back after death is no longer available (/back after teleporting or warping is still available)
v0.1.1 Alpha 11/14/2020
  • Added ViveCraft plugin for VR users
  • Updated permissions
    • /warp is now available
    • /tpa is now available
    • /back is now available
v0.1.0 Alpha 11/8/2020
  • Alpha test of the server begins
